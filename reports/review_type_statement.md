# 综述类型声明

> 生成时间：2026-05-16
> 依据：L21 课件（文献回顾类型地图 + 文献计量型 Review 五特征）

---

## 综述类型定位

**本论文属于：文献计量型文献回顾（Bibliometric Review）**

## 选择理由

| 维度 | 说明 |
|------|------|
| 课程定位 | 文献计量学课程核心写作类型，以计量指标和知识图谱分析领域结构与趋势 |
| 数据特征 | 基于 WoS 643 篇全量文献 + 147 篇 DL 纯净文献，具备文献计量分析的完整数据链 |
| 方法匹配 | 使用 CiteSpace 6.4.1 进行关键词共现、共被引聚类、突现检测，使用 Python bibliometrics-mini 进行网络指标计算 |
| 目标匹配 | 目标是对 Transformer 在半导体制造领域的研究热点、知识基础和前沿趋势进行系统梳理 |

## 五特征满足声明

1. **Review 的结构**：遵循 IMRAD 框架（Introduction-Methods-Results-Discussion-Conclusion）
2. **文献计量的证据**：用 CiteSpace 图谱和网络指标数据支撑论述，而非主观判断
3. **代表文献的解释**：结合 Milestone 论文说明"为什么值得关注"（四准则交叉验证）
4. **作者的主题归纳**：将图谱结果解释为"三层引文结构""三波技术浪潮"等主题
5. **谨慎的趋势总结**：基于突现检测证据指出方向，不写"必将""一定""取代"等强立场预测

## 不属于的类型

| 类型 | 为何不是 |
|------|----------|
| 叙述性综述 | 有文献计量数据和图谱作为证据，非纯叙述梳理 |
| 系统综述 | 未做全文质量评价和 Meta 分析，聚焦图谱叙事 |
| Perspective | 非"大咖式"强立场前瞻，保持证据驱动的谨慎总结 |
